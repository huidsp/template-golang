# template-golang
  
A minimal Go project template.

## Features

- Basic Go module setup
- Example main package
- Ready for Docker and CI integration

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone <repo-url>
   cd template-golang
   ```

2. **Build the project:**
   ```sh
   go build
   ```

3. **Run the project:**
   ```sh
   go run main.go
   ```
